# Project 1 — Titan Company 3-Statement Model & Forecast

**Company:** Titan Company Limited (NSE: TITAN)
**Period:** FY22–FY26 (actuals), FY27E–FY29E (forecast)
**Built in:** Google Sheets

## What this shows
- Fully linked Income Statement, Balance Sheet, and Cash Flow Statement built line by line from source filings
- 3-year forward forecast using revenue growth, margin, and dividend-payout assumptions
- Balance sheet check row confirms Assets = Liabilities across all forecast years
